三消游戏设计原理

五个层次解构 Royal Match 风格界面的视觉密码

构图层次
中心对称 + U形环抱 + 轻上重下
构图核心：棋盘居中占据绝大面积，U形花园路径从底部环抱棋盘，把视线自然收拢到中央。彩色区在上方（轻盈），棕色障碍在下方（厚重），形成稳定的"金字塔"重量分布。中心轴线对称，给人秩序感和安全感。
配色体系
高饱和六色 + 色轮均匀分布 + 面积节奏
配色核心：红黄绿蓝在色轮上等距分布，形成互补+三角色关系，辨识度极高。棕色（低饱和暖色）占据55%面积当"视觉锚点"，让30%的彩色糖果更加珍贵和跳跃。蓝紫底色是冷调中性色，与暖色糖果形成冷暖对比。
光影原理
统一左上光源 + 体积感 + 玩具触感
光影核心：所有元素统一使用左上方光源。左上角有白色高光点，右下为暗面阴影。这种一致的光影赋予了平面图形强烈的3D体积感——红色像果冻、皇冠像金属、盾牌有光泽，产生"想要触摸和滑动"的玩具触感。
视觉引导线
纵向收拢 + 横向前线 + 网格扫描
引导核心：三条引导系统——① 纵向：底部道路向上指引到棋盘中心；② 横向"前线"：彩色区与棕色区的不规则分界线，是可操作边界也是策略焦点；③ 网格：行列结构本身就是最强的扫描引导，让玩家快速寻找匹配组合。
背景 · 边框 · 网格
四层色彩纵深 — 从远到近饱和度递增
层次核心：四层色彩由远到近——天蓝背景 → 金色边框 → 蓝紫网格底 → 高饱和糖果。饱和度逐层递增，形成类似"空气透视"的纵深感。网格线比底色仅深约15%，"隐而不显"——划分空间但绝不喧宾夺主。金色边框同时承担"视觉框架"和"品牌主题（Royal皇家）"的双重功能。
